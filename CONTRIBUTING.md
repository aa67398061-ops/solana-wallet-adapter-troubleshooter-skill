# Contributing

Useful contributions include:

- New error mappings from real Solana wallet failures
- Wallet-specific troubleshooting notes
- Mobile Wallet Adapter test cases
- Next.js provider and SSR edge cases
- Transaction confirmation patterns
- Better QA checklists for production dApps

Before submitting changes:

```bash
npm run validate
```

Keep references concise and actionable. Link to upstream documentation rather
than copying large external docs into this repository.
